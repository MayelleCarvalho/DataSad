Plataforma de Integração de dados de alunos, para avaliação de desempenho escolar.

O Data-SAD é uma aplicação web, desenvolvida para o Instituto Federal do Piauí (IFPI), Campus Teresina Central, para coleta, integração e gerenciamento de dados dos alunos da instituição, disponibilizando informações estatísticas para análise dos profissionais sobre os fatores sociais, demográficos, econômicos e sintomas de ansiedade e depressão por meio de instrumentos(formulários, questionários testes) validados.
